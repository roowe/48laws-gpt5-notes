## 这本书是什么
《权力48法则》（The 48 Laws of Power）是一本围绕“权力如何运作”的策略类作品：它不承诺你变得善良，也不保证你一定成功，它更像是一套对现实博弈的观察与总结。书里反复强调的一点是：权力来自关系与感知，来自信息、联盟、声望与时机；你越能看懂局，越能减少被动。

## 作者信息
罗伯特·格林（Robert Greene）

美国著名公众演讲家和畅销书作家。《纽约客》《时尚先生》的著名编辑和撰稿人。他精通多国语言，曾旅居世界各地，学识广博。1998年，其著作《权力48法则》经由世界著名出版机构企鹅出版公司出版后，即一举荣登畅销书榜首，并持续畅销15年。该书15年来在全球以24种语言版本合计狂销达2000万册。在本世纪初，被美国《财富》杂志推荐为“75本最使人睿智的必读书”之一。其连续多年荣居亚马逊图书网站社科类图书畅销榜榜首。格林的这本《权力48法则》和另外两本书《战略33法则》《征服30法则》被读者命名为“全球智慧书系”。
